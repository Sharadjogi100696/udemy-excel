# Excel

- We can format dates the way we want

```
Jan-2023 --> 01/01/2023
```

- Computation

```
=1000+100+10
=B4+B5+B6+B7
=SUM(B4:B7)
```

- Relative References:

```
=B4/B6
```

- Absolute References:

```
=B4/$B$6 // B6 will be constant
```

## Shortcuts:

- insert a new row

```
CMD + SHIFT + '+'
```

- delete a row

```
CMD + '-'
```
